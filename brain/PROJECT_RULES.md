# Project Rules

## 1. Core Directives

- **Language**: All final answers and documentation must be in Korean.
- **Integrity**: Prioritize data integrity and verification.
- **Brain**: Maintain `brain/` directory for project context.

## 2. Directory Structure

- `brain/`: Project management and context.
- `brain/guidelines/`: Project definitions (e.g., `project_definition.md`).
- `brain/logs/`: Interaction logs and workflow outputs.
